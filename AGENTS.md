# AGENTS.md


## Purpose

This repository is a local study project for building and testing a simple AI agent system. The project is local-only, CLI-only, and single-agent, designed primarily for small Python coding tasks.


## Configuration Rules

Adjustable runtime and agent hyperparameters must live in `config.yaml`.


## Documentation Rules

The root documentation should stay aligned:

- `README.md` explains setup and usage.
- `PROJECT.md` explains the logical design of the whole project.

If the project is updated, the above documentation should also be updated accordingly.


## Style Guide

The code should be PEP8-compliant and follow the Pythonic style guide. Docstrings, comments, and variable names should be clear and descriptive.
